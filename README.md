# DigiTools Platform

A modern digital tools marketplace where users can browse, add tools to their cart, and manage purchases all within a smooth, interactive single-page experience.

## 🛠️ Technologies Used

- **React** — Component-based UI development
- **Vite** — Fast build tool and development server
- **JavaScript (ES6+)** — Core application logic
- **CSS / Tailwind CSS** — Styling and responsive layout
- **DaisyUI** — Tailwind CSS component library for pre-built UI elements
- **React-Toastify** — NPM package for displaying toast notifications
- **JSON** — Static data format used to store and serve product information

## ✨ Features

1. **Cart Management with Notifications** — Add or remove digital tools from the cart with instant toast notifications for every action. Click "Proceed to Checkout" to complete the order, which clears the cart and confirms with a notification.

2. **Live Cart Summary** — The total number of items in the cart is displayed in real time on both the navbar and the Cart tab, while the total price is automatically calculated as you add or remove tools.

3. **Product & Cart Tab Toggle** — Seamlessly switch between the Products tab to browse available tools and the Cart tab to review selected items, all without leaving the page.
